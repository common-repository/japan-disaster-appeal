Adds a badge to the upper left of all pages prompting people to donate to the Japan disaster.

== Description ==

This plugin was created as a method to non-obtrusively display a badge in the upper left of your wordpress web site, which is intended to promote awareness of the recent disaster in Japan.

Hovering your mouse over the badge will reveal links to the donation pages of the three most common crisis web sites, World Vision, Red Cross and All Hands Volunteers.

== Installation ==

Install, activate, done. There is no admin interface for this plugin.

== Uninstall ==

Deactivate the plugin, delete if desired.
